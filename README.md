# Frontend Application

Aplikasi web modern yang dibangun menggunakan React, TypeScript, dan Vite.

## Prasyarat (Requirements)

Sebelum menjalankan aplikasi ini, pastikan Anda telah menginstal:

- **Node.js** (Versi 18 atau lebih baru disarankan)
- **npm** (biasanya terinstall bersama Node.js) atau **yarn** / **pnpm**

## Instalasi

Ikuti langkah-langkah berikut untuk menginstal dependensi proyek:

1.  Clone repository ini atau unduh kode sumbernya.
2.  Buka terminal dan navigasikan ke direktori proyek.
3.  Jalankan perintah berikut untuk menginstal dependensi:

```bash
npm install
# atau
yarn install
# atau
pnpm install
```

## Menjalankan Aplikasi

Untuk menjalankan aplikasi dalam mode pengembangan (development mode), gunakan perintah:

```bash
npm run dev
# atau
yarn dev
# atau
pnpm dev
```

Aplikasi biasanya akan berjalan di `http://localhost:5173`. Periksa terminal untuk alamat yang tepat.

## Membangun untuk Produksi (Build)

Untuk membangun aplikasi agar siap untuk di-deploy ke produksi:

```bash
npm run build
```

Hasil build akan berada di folder `dist`.

## Teknologi Utama

- **Framework**: React
- **Language**: TypeScript
- **Tooling**: Vite
- **Styling**: TailwindCSS
- **State/Query**: TanStack Query
- **Routing**: React Router
